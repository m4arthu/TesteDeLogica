# Como Rodar 
  - Clone este repositorio em sua maquina atravéz da bash do git
    ```````git
      git clone <chave ssh>
    ```````
    ps: chave é gerada da aba codigo -> ssh
    
  - Tenha instalado na sua maquina o  node.js
  - Va para a pasta do  teste que deseja rodar
  - Rode o  seguinte comando:
    ````````powershell
      node index.js
    `````````
